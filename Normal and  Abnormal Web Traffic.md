# Normal and  Abnormal Web Traffic (HTTP/HTTPS)

## Определение
Нормальный веб-трафик - это легитимные HTTP/HTTPS-запросы от пользователей или доверенных систем без признаков автоматизации или атак.


## Основные характеристики
- Источник: реальные IP, браузеры, известные боты
- Частота: нерегулярная, с паузами
- Методы: GET, POST
- Заголовки: корректные (User-Agent, Host)
- URL: логичная структура без аномалий
- Статус-коды: преимущественно 200, 301, 304
- Объём: соответствует типу контента


## Протоколы и порты
- HTTP 80
- HTTPS 443
- DNS 53
- TCP транспортный уровень
- TLS шифрование
- SSH 22
- SMTP 25
- POP3 110
- IMAP 143
- FTP 21 


## Частота запросов

Нормально:
- Пользователь: 1-5 запросов в секунду
- Браузер: кратковременные всплески при загрузке страницы
- Боты: низкая и равномерная частота

Аномально:
- высокий поток запросов с одного IP
- одинаковые интервалы между запросами
- длительная высокая нагрузка


## Объём трафика
- HTML: 10-100 KB
- Изображения: 50-500 KB
- JS/CSS: 10-200 KB
- API: 1-50 KB

Аномально:
- слишком большие ответы без причины
- резкие всплески трафика


## Поведение пользователя

Нормально:
- логичная навигация по сайту
- наличие referer
- паузы между действиями
- активность в дневное время

Аномально:
- хаотичные или нелогичные переходы
- отсутствие пауз
- активность в нетипичное время


## HTTP методы

Нормально:
- GET
- POST

Реже:
- PUT
- DELETE

Аномально:
- использование нестандартных методов без причины
- частые запросы PUT или DELETE


## URL

Нормально:
- /home
- /products/1

Аномально:
- специальные символы в запросах
- попытки обхода путей
- подозрительные параметры


## Ответы сервера

Нормально:
- 200 OK
- 301/302 Redirect
- 304 Not Modified

Аномально:
- большое количество 404
- большое количество 500
- частые 401 или 403


## Сессии

Нормально:
- одна сессия на пользователя
- стабильный IP

Аномально:
- одна сессия с разных IP
- резкая смена географии
- необычно длинные сессии


## User-Agent

Нормально:
- популярные браузеры пример Firefox: 
Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:120.0) 
Gecko/20100101 Firefox/120.0

Аномально:
- пустой User-Agent
- curl, python и другие скрипты пример curl: curl/7.68.0
- поддельные или некорректные -значения пример: Mozilla/5.0 Chrome/999.0


## DNS

Нормально:
- запросы к известным доменам (google.com)
- умеренная частота

Аномально:
- большое количество DNS-запросов
- подозрительные или длинные домены (random-74jfkrj.xyz)


## HTTPS

Нормально:
- валидный сертификат
- современный TLS

Аномально:
- самоподписанные сертификаты
- устаревшие версии TLS/SSL


## География и IP

Нормально:
- ожидаемые регионы
- стабильные IP

Аномально:
- резкая смена страны
- использование VPN или TOR
- множественные IP для одного пользователя


## Baseline

Baseline - это нормальное поведение системы, определённое на основе анализа логов.

Используется для:
- выявления отклонений
- обнаружения атак
